# 模板库

## 一、标题模板（8 套）

### 1. 时效快报型

**公式**：`[刚刚/深夜/实测] [产品名]，[核心卖点/颠覆性事实]`

- 刚刚，智谱开源了他们的最强多模态模型，GLM-4.5V。
- 就在刚刚，可灵更新了，从2世代直接跨越到了3世代。

### 2. 数字反差型

**公式**：`[主体] [做了什么]，[数字化的惊人结果]`

- 他用AI设计丑拖鞋，一年卖了3000万。
- 一个邪修方法，帮你把用Agent的钱省掉80%。

### 3. 对决狂欢型

**公式**：`[动作冲突词]！[A]和[B]同时[事件]，这下真的[狂欢代名词]了`

- 中门对狙！Claude Opus 4.6和GPT-5.3 Codex同时发布，这下真的AI春晚了。

### 4. 终局宣告型

**公式**：`[时间/情绪起手]，是[旧事物]的[终结名词]`

- 明天，是GPT-4o的葬礼。

### 5. 权威定义型

**公式**：`[地域/领域]也有了[绝对形容词]的[产品类别]，他的名字，叫[名称]`

- 中国也有了世界第一的模型，他的名字，叫Seedance 2.0。

### 6. 低门槛承诺型

**公式**：`有手就行，[X]分钟教会你[在哪/用什么]玩转[工具]`

- 全网最详细的Claude Code入门教程，手把手教你玩转AI编程。
- 有手就行，5分钟教会你在Coze上玩转全网爆火的Agent。

### 7. 人文隐喻型

**公式**：`[现象]背后，藏着一场[隐喻名词：战争/革命/黄金时代]`

- AI视频的青年时代，来了。
- 这，才是Vibe Coding的未来。

### 8. 情绪共鸣型

**公式**：`[求求/看到/时隔X天]，[现象]，[情绪反应/引发的思考]`

- 求求AI，别再用"不是…而是…"喂我吃屎了。
- 看到现在的毕业生被AIGC查重折磨，我有话想说。

## 二、开头模板（5 套）

### 1. 极速快报式

```
就在刚刚/深夜，[主角]更新了/来了，更了个大的。
从[旧状态]直接[动作]成了[新状态]。
[一句话定性：这可能是XX领域的转折点]。
```

### 2. 情绪主导式

```
昨天看到一个特别[离谱/炸裂]的事。
[一句话概括核心冲突]。
我必须跟你们聊聊。
```

### 3. 倒计时/宣判式

```
如果不出意外，[具体时间]。
就是[某事物]的[葬礼/终局]了。
道理我都懂，但我还是想写点什么。
```

### 4. 痛点切入式

```
[你是不是也遇到过/最近很多人都在讨论][痛点]？
比如[具体例子]。
这背后，其实是[更大的问题]。
今天我就把[解决方案]写清楚。
```

### 5. 传播背书式

```
我上周写的[内容]，[阅读/转发]到了[数字]。
所以今天把[核心]补齐：手把手写给你。
```

## 三、结尾模板（5 套）

### 1. 行动号召型

```
所以，我的建议是：[具体行动1]，然后[具体行动2]。
现在就开始，别等。

以上，既然看到这里了，点个赞、在看、转发三连，我们下次再见。
```

### 2. 时代升华型

```
[AI技术]的童年时代，结束了。
欢迎来到，[AI技术]的，青年时代。

时代的列车滚滚向前，它不会等任何人。
那就赶紧跑，赶紧跳上去。

以上，既然看到这里了...谢谢你看我的文章，我们，下次再见。
```

### 3. 鸡汤激励型

```
每一次技术革命，都会消灭一些工作，也会创造新的工作。
关键是，[呼吁读者行动]。

以上，既然看到这里了...谢谢你看我的文章，我们，下次再见。
```

### 4. 情绪感悟型

```
回头看这几年，真的太魔幻了。
[感慨技术迭代之快]。
希望大家[美好祝愿]。

以上，既然看到这里了...谢谢你看我的文章，我们，下次再见。
```

### 5. 复盘+窗口期型

```
写在最后：这篇我把[核心内容]写全了。
但[还没测的/后续要补的]，后面再写。

我不知道一年后会怎样，但我知道现在是窗口期。
错过这一波，可能就真的错过了。

以上，既然看到这里了...我们，下次再见。
```
